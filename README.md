## Описание

Это клон сайта PlayStation, выполненный на **React + TypeScript + Vite + TailwindCSS**.  
Проект демонстрирует работу с компонентами, маршрутизацию, стейт-менеджмент и адаптивный дизайн.

![Пример работы](./screenshots/demo.gif)

---

## Технологии

- **React** – библиотека для UI
- **TypeScript** – строгая типизация
- **Vite** – супербыстрая сборка
- **TailwindCSS** – удобная CSS утилитарная библиотека
- **GitHub Actions** – CI/CD, автоматический деплой на GitHub Pages

---

## Установка и запуск локально

Склонируй репозиторий:

```bash
git clone https://github.com/Pavel-Zolenko/playstation-clone.git
cd playstation-clone
npm install
npm run dev
```

Проект будет доступен по адресу: http://localhost:5173/playstation-clone
